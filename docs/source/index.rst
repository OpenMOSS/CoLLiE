CoLLiE: Collaborative Tuning of Large Language Models in an Efficient Way
=========================================================================

`CoLLiE <https://github.com/OpenLMLab/collie>`_ 是一款用于高效训练、微调大模型的工具包，支持数据并行、张量并行、流水线并行、ZeRO优化等多种优化方式。
